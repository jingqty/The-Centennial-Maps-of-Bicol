# The Centennial Maps of Bicol

These maps took a little over a day to complete—most of that time was spent figuring things out along the way. I was inspired after reading about some of the oldest vintage maps of the Philippines, which made me curious about how mapping was done before modern tools. It’s fascinating how accurate these early maps were, even without a bird’s-eye view.

The maps were sourced from various online archives and digital libraries, then cleaned and edited using GIMP. They were georeferenced and aligned using the Georeferencer plugin in QGIS (v3.34.9 Prizren).\
